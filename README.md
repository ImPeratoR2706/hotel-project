Отчёт о реализации практик. Тема проекта: Отель.

Практическое занятие №1: CSS-препроцессоры.

Я использовал препроцессор SASS. В моём проекте он был задействован для изменения цвета переменных (таких как $hotel-primary, $hotel-accent и т.д.).Я использовал его для стилизации кнопки бронирования с hover-эффектом.

Практическое занятие №2: Сервер на Node.js + Express.

Я использовал платформу Node.js и фреймворк Express. В моём проекте они были задействованы для разработки серверной части API. Я использовал их для реализации CRUD операций управления номерами отеля (просмотр, добавление, редактирование, удаление), где объект номера содержит поля id, название и стоимость.

Практическое занятие №3: JSON и внешние API

Я использовал инструмент Postman и формат JSON. В моём проекте они были задействованы для тестирования API и работы с внешними данными. Я использовал их для проверки маршрутов сервера отеля (не менее 3 запросов) и получения данных о погоде через внешний API (не менее 5 запросов).

Практическое занятие №4 API + React

Я использовал библиотеку React и клиент Axios. В моём проекте они были задействованы для разработки клиентской части приложения. Я использовал их для создания интерфейса управления номерами отеля (просмотр, создание, редактирование, удаление) с связью с серверным API.

Практическое занятие №5 Расширенный REST API

Я использовал инструменты Swagger. В моём проекте они были задействованы для документирования серверного API. Я использовал их для автоматической генерации интерактивной документации всех CRUD-операций управления номерами отеля через JSDoc-аннотации

Практическое занятие №7 Базовые методы аутентификации (Хеширование паролей)

Я использовал библиотеку bcrypt. В моём проекте она была задействована для безопасной аутентификации пользователей. Я использовал её для реализации регистрации и входа в систему отеля с хешированием паролей и хранением их в защищённом виде.

Практическое занятие №8 Базовые методы аутентификации (JSON Web Token)

Я использовал технологию JWT. В моём проекте она была задействована для Stateless аутентификации. Я использовал её для выдачи access-токенов при входе и защиты маршрутов управления номерами отеля от неавторизованного доступа.
